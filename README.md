# chilika-wildlife-sanctuary
🌿 Complete single-file website showcasing CHILIKA LAKE - Asia's largest brackish water lagoon & Ramsar Wetland of International Importance.

🐬 Home to endangered Irrawaddy Dolphins, 150+ migratory birds, 52 fish species & rich biodiversity.

✨ Features:
- Fully responsive design (Mobile-first)
- Smooth animations & scroll effects
- Interactive wildlife cards
- Conservation programs showcase
- Google Maps integration
- No external dependencies - Single HTML file!

🎯 Perfect for: Wildlife education, ecotourism promotion, portfolio showcase, conservation awareness.
